# ✈️ Wanderlog - Digital Travel Journal

Wanderlog is a web-based digital travel journal where users can log, track, and manage their travel memories and itineraries.

## 🚀 Key Features
- **Add Travel Logs:** Easily log trips with details like Title, Destination, Date, Cover Image URL, and Notes.
- **Local Storage Integration:** All added trips are automatically saved in the browser so data isn't lost on refresh.
- **Delete Functionality:** Remove old or test trips with a single click.
- **Empty State Toggle:** Displays a helpful friendly message when no trips are saved.
- **Smooth Navigation:** Interactive UI with smooth scrolling to sections.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure and form elements
- **CSS3:** Responsive layout and styling
- **JavaScript (ES6):** Dynamic rendering, DOM manipulation, and LocalStorage management
